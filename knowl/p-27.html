The definition for the roots given in Definition part (b) is chosen as a consequence of Theorem property (c). That is, a root \(\sqrt[k]{x}\) has the meaning that when this number is raised to the power \(k\text{,}\) we obtain the value \(x\text{.}\) If we wanted to write that root as a power \(\sqrt[k]{x} = x^r\) where we didn't know \(r\) in advance, property (c) would require

\begin{equation*} x = (\sqrt[k]{x})^k = (x^r)^k = x^{rk}. \end{equation*}

In order for this to be true, we require \(rk = 1\) so that \(r=\frac{1}{k}\text{.}\)
